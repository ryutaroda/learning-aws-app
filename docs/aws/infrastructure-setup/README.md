# Learning環境 AWS構築手順

## 概要
学習用のAWS環境を段階的に構築するための手順書です。

## 作業手順

### 1. ネットワーク基盤の構築
- [x] **VPC設計・構築** → [vpc.md](./1_vpc.md)
  - VPC、サブネット、ゲートウェイ、ルートテーブルの設計・構築
- [ ] **NATインスタンス作成** → [1_2_nat-instance.md](./1_2_nat-instance.md)
  - NATゲートウェイをNATインスタンスに置き換えてコスト削減

### 2. データベース基盤
- [x] **RDS設計・構築** → [2_rds.md](./2_rds.md)
  - PostgreSQL RDS、セキュリティグループ、サブネットグループの構築

### 3. 環境変数設定
- [ ] **環境変数設定** → [3_secrets-management.md](./3_secrets-management.md)
  - S3バケット（非機密情報）、Secrets Manager（機密情報）

### 4. セキュリティ設定
- [ ] **セキュリティグループ設計・作成**
  - Web層、アプリ層、DB層のセキュリティグループ

### 5. コンピューティングリソース
- [ ] **EC2インスタンス作成・配置**
  - Webサーバー、アプリケーションサーバーの構築

### 6. 運用・監視
- [ ] **CloudWatch設定**
- [ ] **バックアップ設定**

## 環境情報
- **環境名**: learning-stg
- **リージョン**: ap-northeast-1
- **使用AZ**: 1a, 1c
